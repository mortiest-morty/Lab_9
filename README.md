# Lab_9

ЛАБОРАТОРНАЯ РАБОТА №9

File12. Дан файл целых чисел. Создать два новых файла, первый из которых содержит четные 
числа из исходного файла, а второй — нечетные (в том же порядке). Если четные или нечетные 
числа в исходном файле отсутствуют, то соответствующий результирующий файл оставить пу
стым. 

File35. Дан файл целых чисел, содержащий менее 50 элементов. Увеличить его размер до 50 эле
ментов, записав в начало файла необходимое количество нулей. 

File54. Дана строка S и файл-архив целых чисел, содержащий данные из нескольких (не более 
шести) файлов в формате, описанном в задании File52. Для каждого из файлов, содержащихся в 
архиве, найти среднее арифметическое всех его элементов (вещественное число) и записать 
найденные числа (в том же порядке) в файл вещественных чисел с именем S.
